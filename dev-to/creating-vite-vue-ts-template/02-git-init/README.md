# Git init

## Creating git repo

1. `mkdir vue-ts && cd vue-ts`
1. `git init`
1. `git commit --allow-empty -m 'init'`
1. `git remote add origin git@github.com:imomaliev/vue-ts.git`
1. `git push -u origin HEAD`
